# WeatherApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.2.4.

##Dependencies
Angular `6.1.0`

NPM version `6.4.1`

Node version `8.12.0`

CLI - version `6.2.4`

## Deploy project

Run `npm install` for install dependencies. 

Run `npm run start` for build and start application in browser.

Run `npm run build` for build project in production mode. Source files will be in `dist/weather-app` folder.
